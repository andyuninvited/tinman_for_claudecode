TinMan

Scheduled heartbeat for Claude Code. Runs your HEARTBEAT.md checklist through Claude on a timer and tells you when something needs attention.

  • Plain-English HEARTBEAT.md checklist
  • macOS launchd + Linux cron
  • Notify-only by default
  • sane / paranoid / chaos presets
  • Logs every beat, rotates automatically
  • Zero external dependencies
pip install tinman-for-claudecode GitHub ‚Üí
üì°ü§ñ

C3Poh

Telegram bridge for Claude Code. DM your agent from anywhere, get answers back. Heartbeat alerts from TinMan land on your phone automatically.

  • Long-polling (no inbound ports needed)
  • Allowlist-only DM policy by default
  • Token never written to disk
  • Local notify server for TinMan
  • Group chat support with @mention
  • Zero external dependencies
pip install c3poh-for-claudecode GitHub ‚Üí

Security presets

Pick the one that matches your comfort level. You can always change it.
